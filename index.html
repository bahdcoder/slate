    Introduction

    Welcome to the Mentor-Temple API, A learning management system for matching developers and mentors in the community.

    We have language bindings in JavaScript. You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Authentication

    In order to use the API, you should authenticate your request by including the user JWT as a bearer token value:

    Authorization: Bearer JWT_HERE

    URI

    Mentor-Temple API is hosted on the following base URI: https://api.mentortemple.io

  "email": "amanda@dyners.co",
  "password": "secret",
  "account_type": "developer"

    This endpoint retrieves registers mentors and developers on the platform.

    HTTP Request

    POST /users/register

    Parameters

    Parameter Description
    name The name of the user.
    email The email of the user.
    password The password for this user.
    account_type The account type for the user being registered. Can be of two types: developer and mentor.

    This endpoint generates a JWT for a user.

    HTTP Request

    POST /users/login

    Parameters

    Parameter Description
    email The email of the user
    password The password of the user
